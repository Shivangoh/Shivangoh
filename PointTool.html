Point

A four-step approach to evaluating an option, idea, solution, etc. "POINt" is an acronym for Pluses, Opportunities, Issues, and New thinking.

How to use it:

  1. Make a list of the pluses.
  2. Then make a list of the opportunities (potentials).
  3. Then make a list of the issues (concerns, expressed as problem statements).
  4. Finally make a list of new thinking on how to overcome the issues.
